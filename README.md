# tictactoell

A simple Tic-Tac-Toe console game implemented in Haskell


# Basic Haskell

### Stack

    λ → stack build
    λ → stack test
    λ → stack ghci
    λ → stack exec tictactoell-exe

### About GHCI

Examples of how to use GHCI when executing `stack ghci`:

    Prelude> :load Main

More about GHCI [here](http://learnyouahaskell.com/starting-out)
